Welcome

What is Auto Articles?

Auto Articles helps you plan, draft, and publish SEO-ready posts to WordPress—with AI doing the heavy lifting so you spend minutes instead of hours on each article.

  • Generate long-form content from your briefs and prompts
  • Publish and schedule directly to your site
  • Keep tone, keywords, and internal links consistent at scale

How to get started

Connect your WordPress site, then layer prompts and context links so every article matches your brand.

  1. a
    Project

    Click Add project above to name your site and add its URL. You can add more brands or sites anytime from this workspace.

  2. b
    WordPress connector

    After you open a project, use Tools from the sidebar to download the WordPress plugin ZIP so Yoast meta and REST work securely with Auto Articles.

    WordPress Admin → Plugins → Add New → Upload Plugin → Install → Activate.

  3. c
    Application password

    In WordPress, open your user profile and create an Application Password. Paste it with your username into Project settings on that project (not your regular login password).

    Tip: open Project settings from the project header after you create a project.

Article writing prompts

Once your site is connected, add at least one writing prompt: it tells the AI how to structure tone, sections, and SEO focus for each article.

Open Tools from the sidebar → Writing prompts, or use Configuration. Set a default prompt so “Generate” always knows your house style.

Featured image prompts

Image prompts describe the hero image you want—layout, mood, and how it should relate to the article title and keywords.

Add them under Tools → Image prompts. The model uses your prompt to produce on-brand featured images for each post.

Context links

Context links map phrases to URLs on your site. When articles are generated, those phrases can be linked automatically—great for internal linking and pillar pages.

Manage them from Tools → Context links. Add the phrase once; reuse it across many articles.
